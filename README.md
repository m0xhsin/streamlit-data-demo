**Interactive Data Visualization Demo** showcases the power of **Streamlit** for creating dynamic dashboards. Explore interactive charts, tables, metrics, images, widgets, and real-time calculations—all in one engaging interface.

---

## 🚀 Features
- 🖼️ Display and interact with images  
- 📊 Interactive data tables and filtering  
- 📈 Bar charts and metrics visualization  
- 🔮 Predictive calculations with sliders  
- 🌏 Select options with radio and multiselect widgets  
- ⏳ Progress bar simulation for live updates  
- Fully interactive **Streamlit dashboard**  

---

## 🛠️ Tech Stack
- Python 3  
- Streamlit  
- Pandas  
- Matplotlib / Streamlit charts
